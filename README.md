# Požega Tourism Portal & Knowledge Base

This is a modern, high-performance web portal and knowledge base for the city of Požega, Croatia. It serves as a digital archive for the Požega Tourist Board.

## 🚀 Features
- **Modern Design**: Premium Glassmorphism UI built with Vanilla CSS.
- **Fast Performance**: Powered by Next.js 15 (App Router).
- **Markdown Driven**: Content is managed as Markdown files for easy archival and editing.
- **Exhaustive Knowledge**: Includes detailed pages on Attractions, Gastro partners, News archives (2025-2026), and Official documents.

## 🛠️ Tech Stack
- **Framework**: Next.js
- **Styling**: Vanilla CSS (Custom Glassmorphism)
- **Content**: Markdown + Gray-matter
- **Deployment**: Vercel

## 📁 Content Structure
All tourism data is stored in the `/content` directory, organized by category:
- `/Attractions`: Sights and monuments.
- `/Gastro`: Restaurants, Wineries, and OPGs.
- `/News`: News archive 2025-2026.
- `/Official`: Acts, regulations, and assembly sessions.
- `/Travel`: Logistics, arrival, and parking.
